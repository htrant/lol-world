# lol-world
Attempt to examine [LOLCODE](http://lolcode.org/).

## Install
```
$ git clone https://github.com/justinmeza/lci.git
$ cd lci
$ cmake .
$ make && make install
```

## Programs
#### Hello world
```
$ lci hello.lol		# print out HELLO WORLD!
```
